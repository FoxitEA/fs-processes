Fs.Processes

The Fs.Processes library provides a set of .NET class wrappers for the Windows CreateProcess and Windows Job Objects APIs. The goal of this library is to provide comprehensive coverage of the CreateProcess API. The library is not intended to replace the System.Diagnostics.Process class.
